#### 투자자의 마음가짐

* 전쟁 장기화에 따라 어떻게 기업에 투자해야 하는가?
  * 투자한 기업이 돈을 버는 능력을 여전히 가지고 있는가?
  * 이란 전쟁이 단기전인가 장기전인가? 원유가 5% 부족하면 유가는 얼마나 50%~500% 변화할 수 있다
  * 거시경재 (금리/환율)의 변화에 따라 어떻게 변화하는가?
  * 정책/지원 (AI 반도체 등)이 어떻게 변화하는가?
  * 어떤 산업이 이득을 보고, 손해를 보는가? 원유는 카타르 중동만 막히지 러시아, 미국, 캐나다 등은 여전히 문제가 없다
    * 중동에서 원유를 받아서 처리하는 아시아(중국, 일본, 한국 등)은 영향을 본다
* 이제 금리 인하는 멈추고 **금리 상승의 초입부에 있으며 이에 따른 종목 리밸런싱 및 대비**가 필요하다
  * 기존 주도주 (IT, 빅테크, S/W 등) 는 하락 사이클
  * **전쟁 및 물가관련 수혜주**가 상승 사이클
* 고유가가 오래 유지되는 상황에서 만약에 금리 인하를 하는 경우
  * 유럽/호주/일본은 금리 인상이 예정, **미국/한국만 금리 하락** 한다면
  * 환율상승, 수입물가 상승, 채권금리 상승 및 대출이자 증가는 경기 둔화가 예상됨
  * 미국은 유가 상승을 막기위한 모든 일을 하려고 하지만 어렵다

####  용어 및 도메인 이해

* HBM4 vs. DDR7

  * HBM4 : GPU 성능을 살리는 “핵심 메모리”로 광대역폭의 메모리로 GPU 에 붙어서 AI 성능 강화 핵심 LLM 추론 엔진

  * DDR7 : AI를 “확장 가능하게 만드는 메모리” 상대적으로 낮은 대역폭의 가격이 저렴하고 CPU 에 붙어서 롱텀 메모리

  * HBM4 는 캐시의 역할이고 DDR7 은 메인 메모리에 가까운 장치이다

  * ```
            [ NVMe / Network ]
                     │
                ┌─────────┐
                │   CPU   │
                └─────────┘
                     │
                   [ DDR ]
                     │
         ┌───────────┼───────────┐
         │           │           │
       GPU1        GPU2        GPU3
         │           │           │
       HBM         HBM         HBM
         └────── NVLink ────────┘
    ```

#### 어떤 마인드로 접근하면 좋을까?

* 국내 주식

  * 에너지 관련 이미 올라간 주식은 조정을 받는 경우 진입

    * 저는 단기적으로 많이 올랐을 경우 조정이 올 때 기회가 오면 사면 좋고 아니면 보내는 전략을 씁니다. 좋은 주식은 많고, 꼭 사야하는 것은 아

      닙니다. 좋은 기회가 오지 않으면 내것이 아니라는 생각으로 흘리는 것도 필요합니다.

  * 전쟁이 장기화됨에 따라 하나는 원유공급이고, 하나는 가스공급입니다

    * 미국의 천연가스 관련 회사와 장기계약이 있을 수 있음
    * 3대 원유거래소는 영국의 브렌트유, 미국의 서부텍사스유, 중동의 두바이유
